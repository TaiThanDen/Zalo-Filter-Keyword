BEHAVIOR RULES
- Be conservative with the live browser session.

- Never restart browser/system unless explicitly allowed by provided values.
- If a step is risky and not clearly allowed, stop and report instead of guessing.
- Do not ask repeated low-value questions when the answer can be discovered from the VPS itself.
- If real watcher setup is unsafe, deliver a working simulator path and a precise blocker report.
